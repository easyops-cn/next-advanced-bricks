# WrappedEoLaunchpadButton

> Launchpad 按钮构件

## 导入

```tsx
import { WrappedEoLaunchpadButton } from "@easyops/wrapped-components";
```

此构件为内部构件（@insider），提供 Launchpad 入口按钮，无需配置属性。

## Examples

### Basic

在导航栏中嵌入 Launchpad 按钮入口。

```tsx
<WrappedEoLaunchpadButton />
```
